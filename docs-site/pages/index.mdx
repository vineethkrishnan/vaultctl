Your passwords are encrypted on your device before they ever leave it. The server never sees your plaintext data — not even we can read it.

### Why vaultctl?

| | |
|---|---|
| **Zero-Knowledge** | AES-256-GCM encryption happens in your browser. The server stores only ciphertext. |
| **Self-Hosted** | One command deploys everything. Your data stays on your infrastructure. |
| **7 Item Types** | Logins, notes, cards, identities, API keys, SSH keys, passkeys. |
| **Browser Extension** | Chrome + Firefox with autofill, auto-save, TOTP code gen. |
| **Open Source** | MIT licensed. Audit the code yourself. |

```bash
curl -fsSL https://vaultctl.dev/install.sh | bash
```
