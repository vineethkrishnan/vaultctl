VaultCTL Complete Light/Dark Asset Pack

Structure:
- /light contains black assets for light themes.
- /dark contains white/inverted assets for dark themes.
- Each theme contains:
  svg/
  favicon/
  app-icons/
  android/
  ios/
  web/
  logo/

SVG files are standalone and self-contained. They embed cleaned transparent PNG artwork because no original vector source file was available.
