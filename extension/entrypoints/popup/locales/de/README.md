# German (de) translations - machine-drafted

The strings in `popup.json` and `common.json` are machine-drafted and pending
review by a native speaker. Conventions used:

- Formal address ("Sie", not "du").
- "Tresor" for vault, "Zugangsdaten" for credentials, "Master-Passwort" for the
  master password.

Please review for tone and terminology before treating these as final.
